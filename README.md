- 1주일에 무조건 알고리즘 3문제 이상 풀고 본인의 폴더에 올릴 것.

  - 난이도 무관.

  - 꽃은 기프티콘으로, 한 명당 1만원 이상의 꽃 선물.
- 한 명 이하만 남을 때까지 계속된다.
